# 3D Pong

This is a 3D Pong game made with OpenGL. Consists of two players and basic
controls for the game.

![preview](img/preview.png)

## Controls

Key   | Description
---------|----------
`W`      |  Up (Player 1)
`S`      |  Down (Player 1)
`Arrow Key Up` | Up (Player 2)
`Arrow Key Down` | Down (Player 2)
`P` | Pause
`I` | Start
`R` | Reset
